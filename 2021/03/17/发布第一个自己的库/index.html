学习一下发布库

前言

发布依赖库有两种方式：

  • 发布到 jitpack

  • 发布到 jcenter

两种发布方式的过程并不一样，且发布后两者的添加方式也不一样：除使用 implemention 引用库以外，前者还需要在 build.gradle 中添加 maven { url “jitpack.io” }。

个人比较喜欢使用依赖库的时候简单一些，所以本文记录的是后者的发布方法。

准备工作

Android Studio、开源项目、Bintray账号（因仓库jcenter属于Bintray）

申请 BinTray 账号

可用 Github 或 Google 账号 sign in 后填写信息注册。但我在注册的时候使用 Github 遇到了问题，只能转而选择 Google 账号。
